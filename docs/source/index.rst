Aplikacja Ankiety
========

Aplikacja Ankiety służy do przeprowadzania anonomowych ankiet z zachowaniem możliwości sprawdzenia przez ankietowanego czy jego
odpowiedź w niezmienionej formie znajduje sie w bazie danych oraz sprawdzenia czy dana osoba udzieliła odpowiedzi.


Cechy programu
--------

- tworzenie ankiet i zbieranie odpowiedzi
- zachowanie anonimowości głosujących
- możliwośc sprawdzenia czy dana osoba udzieliła odpowiedzi


Instalacja
------------

Aplikacja działa na serwerze Django.
